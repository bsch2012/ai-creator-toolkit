# Caption Forge - Sample Templates

Instagram/TikTok optimized captions for AI content.

## Hook Templates

### Tech/AI News
```
This new AI is insane...
[visual hook]
[reveal]
```

### Tutorial/How-to
```
How to make [X] with AI:
Step 1: [action]
Step 2: [action]
Done. Easy.
```

### Behind-the-Scenes
```
AI video creation process:
Real footage vs AI result
[comparison]
```

## CRT Character Captions

### CRT Signal (AI News)
```
Breaking: [news headline]
[quick breakdown]
Link in bio.
```

### CRT Static (Creator Tools)
```
New tool: [tool name]
- Feature 1
- Feature 2
- Feature 3
Worth it? [verdict]
```

## Format Rules
- Plain text only (no smart quotes/dashes)
- Keep under 150 characters for IG
- Hook in first 3 words
- CTA at end (optional)
