# InVideo AI Quick Start Guide

Text-to-video production in under 5 minutes.

## What is InVideo AI?

InVideo AI transforms text prompts into complete, publish-ready videos with:
- AI-generated scripts
- Stock footage selection (16M+ assets)
- Voiceover generation (50+ languages)
- Music & transitions
- Magic Box editing (natural language commands)

## Quick Start

### 1. Create Account
- Visit: ai.invideo.io
- Sign up (free plan available)
- 10 min/week AI generation on free tier

### 2. Generate Your First Video

**Prompt Structure:**
```
Create a [duration] [platform] video about [topic].
Tone: [casual/professional/fun]
Structure: [hook/problem-solution/tutorial]
Style: [aesthetic preference]
Include: [specific elements]
End with: [CTA]
```

**Example Prompt:**
```
Create a 30-second TikTok video about AI video generation tools.
Tone: Casual and energetic
Structure: Hook → 3 key features → CTA
Style: Modern, fast-paced with trendy transitions
Include: Text overlays highlighting each feature
End with: "Link in bio to try it free"
```

### 3. Edit with Magic Box

Natural language commands:
- "Change voiceover to British accent"
- "Make intro more exciting"
- "Remove scene 2"
- "Add upbeat background music"
- "Shorten to 20 seconds"

## Pro Tips

### Prompt Engineering
✅ **Be specific** - Include duration, platform, tone
✅ **Structure matters** - Hook/body/CTA performs best
✅ **Visual direction** - Describe desired aesthetic
❌ **Avoid vague** - "Make a cool video" won't work well

### Best Use Cases
- **Social media ads** (15-60s)
- **Tutorial intros** (quick hooks)
- **Product demos** (feature highlights)
- **Content repurposing** (blog → video)

### Limitations
⚠️ **Not ideal for:**
- Long-form content (10+ minutes)
- Highly specific brand guidelines
- Frame-accurate editing needs
- Complex animations

## Workflow Integration

### InVideo + Kling AI
1. Generate custom clips with Kling AI
2. Upload to InVideo as custom media
3. Use InVideo for script + voiceover + editing
4. Export final production

### InVideo + Hedra
1. Create character animations in Hedra
2. Import to InVideo timeline
3. Add context footage + voiceover
4. Polish with Magic Box edits

### InVideo Standalone
1. Text prompt → Full video
2. Magic Box refinements
3. Export → Publish
4. Fastest workflow for generic content
